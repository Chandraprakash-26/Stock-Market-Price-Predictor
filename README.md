#  Stock Market Price Predictor

The **Stock Market Price Predictor** is a Machine Learning based web application designed to analyze historical financial data and predict stock price trends. The system uses regression techniques to estimate the **next-day closing price** and provide simple investment suggestions.

## Project Overview

* Developed Machine Learning models to predict **stock price trends using historical financial datasets**
* Applied **data preprocessing and regression algorithms** to analyze market behavior
* Provides **short-term price forecasting** based on historical data patterns

## Features

* Predicts **next-day stock closing price**
* Generates **Buy / Sell / Hold recommendations**
* Supports **multiple international and Indian stocks**
* Allows **custom stock ticker input**
* Simple and user-friendly web interface

## Technologies Used

Python, Flask, Pandas, NumPy, Scikit-learn, Matplotlib, HTML, CSS, Bootstrap, yFinance API
