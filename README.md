# Stock-Market-Price-Predictor

A Machine Learning based web application that predicts **stock price trends** using historical financial data. The system fetches stock data from Yahoo Finance and predicts the **next day's closing price** using a regression model.

## Project Overview

* Developed ML models to predict **stock trends using historical financial datasets**
* Applied **data preprocessing, visualization, and regression algorithms**
* Achieved accurate forecasting for **short-term price movements**

## Technologies

**Python, Flask, Pandas, NumPy, Scikit-learn, Matplotlib, HTML, CSS, Bootstrap, yFinance API**

## Features

* Predicts **next-day stock price**
* Provides **Buy / Sell / Hold recommendation**
* Supports **multiple international and Indian stocks**
* Allows **custom ticker input**

## How to Run

1. Clone the repository

git clone https://github.com/Chandraprakash-26/Stock-Market-Price-Predictor.git

2. Install dependencies

pip install flask yfinance pandas numpy scikit-learn matplotlib

3. Run the application

python app.py

4. Open in browser

http://127.0.0.1:5000/
